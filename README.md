# Dotfiles

Install with

	./install

## Neovim dependencies for this setup

Install plugins with

	:PlugInstall

coc.nvim requires node

	brew install node

neovim 0.5

	brew install neovim --head

trash-cli

	brew install trash-cli

universal-ctags

	brew install --HEAD universal-ctags/universal-ctags/universal-ctags
